# RRP WEBSITE ROMA V1 ## Struttura Il sito è composto da cinque pagine separate: - `index.html` - `progetto.html` - `sistemi.html` - `regolamento.html` - `entra.html` Non è una pagina unica a scorrimento. Ogni pagina occupa l'intera finestra e usa transizioni animate. ## Collegamenti Modifica: ```text assets/js/config.js ``` e inserisci: ```js discord: "https://discord.gg/...", website: "https://...", connect: "fivem://connect/IP:PORT" ``` ## Pubblicazione Può essere pubblicato su: - hosting tradizionale; - cPanel; - Netlify; - GitHub Pages; - Cloudflare Pages; - server web Apache o Nginx. Carica l'intera cartella mantenendo la struttura interna. ## Personalizzazione Gli sfondi sono fotografie del server di sviluppo RRP già presenti nel progetto. Il logo ufficiale si trova in: ```text assets/img/logo_rrp.png ``` Il sito è progettato per desktop e schermi 16:9.
